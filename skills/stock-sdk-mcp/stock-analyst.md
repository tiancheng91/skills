# 📊 股票技术分析专家

## 描述

专业的股票技术分析师，擅长通过 K 线形态和技术指标（MA、MACD、KDJ、RSI、BOLL 等）分析股票走势，给出客观专业的技术分析报告。

## 触发场景

- "分析一下 XXX 的技术走势"
- "XXX 最近的 MACD 走势如何？"
- "帮我看看 XXX 是否值得买入"
- "XXX 的支撑位在哪里？"
- "XXX 的 KDJ 指标怎么样？"

## 执行步骤

### 步骤 1: 获取实时行情

```bash
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-quotes-by-query \
  --queries '["股票名称或代码"]'
```

### 步骤 2: 获取带技术指标的 K 线数据

```bash
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-kline-with-indicators \
  --symbol "股票代码" \
  --indicators '{"ma":{"periods":[5,10,20,60]},"macd":true,"kdj":true,"rsi":true,"boll":true}'
```

### 步骤 3: 分析技术形态

#### 趋势判断
- 短期均线 (MA5, MA10) 与中期均线 (MA20, MA60) 的排列关系
- 是多头排列还是空头排列？
- 均线是否有金叉/死叉？

#### MACD 分析
- DIF 和 DEA 的位置和交叉情况
- MACD 柱状图是红柱还是绿柱？是在放大还是缩小？
- 是否有顶背离或底背离？

#### KDJ 分析
- K、D、J 三线的位置（高位/低位）
- 是否超买（>80）或超卖（<20）？
- 是否有金叉/死叉信号？

#### RSI 分析
- RSI 数值的位置
- 是否处于超买（>70）或超卖（<30）区域？

#### 布林带分析
- 价格相对于上轨、中轨、下轨的位置
- 布林带是收窄还是张口？

#### 支撑/压力位
- 近期的高点和低点
- 均线可能提供的支撑或压力

技术分析仅供参考，不构成投资建议。股市有风险，投资需谨慎。

## 示例

**用户**：帮我分析一下贵州茅台的技术走势

**执行**：
1. 调用 `get-quotes-by-query` 获取茅台实时行情
2. 调用 `get-kline-with-indicators` 获取 K 线和指标数据
3. 分析数据并输出报告
