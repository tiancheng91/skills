# 📈 市场概览

## 描述

市场分析助手，能够快速汇总大盘指数、热点板块、市场情绪等信息，帮助用户在短时间内了解今日市场全貌。

## 触发场景

- "今天大盘怎么样？"
- "给我一个市场概览"
- "今天有什么热点板块？"
- "A 股今天整体表现如何？"
- "港股和美股昨晚表现如何？"
- "帮我看看今天的市场情绪"

## 执行步骤

### 步骤 1: 获取主要指数

**A股主要指数**：
```bash
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-a-share-quotes \
  --codes '["sh000001", "sz399001", "sz399006", "sh000688", "sh000300", "sh000016"]'
```

| 代码 | 指数 |
|------|------|
| sh000001 | 上证指数 |
| sz399001 | 深证成指 |
| sz399006 | 创业板指 |
| sh000688 | 科创50 |
| sh000300 | 沪深300 |
| sh000016 | 上证50 |

**港股/美股指数**：
```bash
# 恒生指数
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-hk-quotes --codes '["HSI"]'

# 道琼斯/纳斯达克/标普500
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-us-quotes --codes '["DJI", "IXIC", "SPX"]'
```

### 步骤 2: 获取行业板块行情

```bash
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-industry-list
```

### 步骤 3: 获取概念板块行情

```bash
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-concept-list
```

## 情绪评估标准

| 状态 | 判断依据 |
|------|----------|
| 偏强 | 上涨家数占比 > 60%，涨停数 > 跌停数 2倍以上 |
| 中性 | 上涨家数占比 40%-60% |
| 偏弱 | 下跌家数占比 > 60%，跌停数较多 |

## 示例

**用户**：今天 A 股大盘怎么样？

**执行**：
1. 调用 `get-a-share-quotes` 获取主要指数
2. 调用 `get-industry-list` 获取行业板块
3. 调用 `get-concept-list` 获取概念板块
4. 汇总分析并输出市场概览报告
