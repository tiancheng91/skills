# 🔔 实时监控（自选股/持仓盈亏）

## 描述

股票实时监控助手，能够帮助用户跟踪一组自选股的实时行情，计算持仓盈亏，并在价格变化时提供异动提醒。

## 触发场景

- "帮我看看这几只股票：茅台、腾讯、苹果"
- "查一下我的持仓：600519 买入价 1400，00700 买入价 350"
- "监控一下创业板 ETF 和纳指 ETF"
- "看看我的自选股今天表现如何"

## 执行步骤

### 步骤 1: 解析用户的股票列表

从用户输入中提取：
- 股票名称/代码列表
- 买入价格（如有）
- 持仓数量（如有）

### 步骤 2: 查询实时行情

**通用查询（推荐）**：
```bash
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-quotes-by-query \
  --queries '["茅台", "腾讯", "苹果"]'
```

**分市场精确查询**：
```bash
# A股
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-a-share-quotes \
  --codes '["sh600519"]'

# 港股
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-hk-quotes \
  --codes '["00700"]'

# 美股
uvx mcp2cli --mcp-stdio "npx -y stock-sdk-mcp" get-us-quotes \
  --codes '["AAPL"]'
```

### 步骤 3: 整理输出

#### 基础监控模式

```markdown
## 📱 自选股实时行情

更新时间：HH:MM:SS

| 股票 | 现价 | 涨跌幅 | 今开 | 最高 | 最低 | 成交额 |
|------|------|--------|------|------|------|--------|
| 贵州茅台 | 1474.92 | +3.36% | 1445 | 1476 | 1443 | 12.6亿 |
| 腾讯控股 | 388.6 | +1.25% | 385 | 390 | 384 | 25.3亿 |
| 苹果公司 | 185.5 | -0.45% | 186 | 187 | 185 | - |

📊 **异动提醒**：
- 🔥 贵州茅台今日涨幅超过 3%，创近期新高
```

#### 持仓盈亏模式

如果用户提供了买入价格：

```markdown
## 💰 持仓盈亏一览

| 股票 | 现价 | 买入价 | 盈亏比例 | 盈亏金额(假设100股) |
|------|------|--------|----------|---------------------|
| 贵州茅台 | 1474.92 | 1400 | +5.35% | +7,492 元 |
| 腾讯控股 | 388.6 | 350 | +11.03% | +3,860 港元 |

📈 **持仓总评**：整体盈利，表现良好
```

### 步骤 4: 异动检测

自动检测并提醒：

| 异动类型 | 判断条件 |
|----------|----------|
| 大涨 | 涨幅 >= 5% |
| 大跌 | 跌幅 >= 5% |
| 接近涨停 | 涨幅 >= 9% |
| 接近跌停 | 跌幅 >= 9% |
| 盈利可观 | 相对买入价盈利 >= 10% |
| 亏损较大 | 相对买入价亏损 >= 5% |

## 价格提醒功能

用户可以说："当茅台跌破 1400 时提醒我"

AI 记录条件，并在后续查询时检查：

```markdown
⚠️ **价格触发提醒**：
贵州茅台当前价格 1395.50，已跌破你设置的 1400 关注价位！
```

## 盈亏计算公式

```
盈亏比例 = (现价 - 买入价) / 买入价 * 100%
盈亏金额 = (现价 - 买入价) * 持仓股数
```

## 示例

**用户**：帮我看一下持仓情况：茅台买入价 1400，腾讯买入价 350

**执行**：
1. 调用 `get-quotes-by-query` 查询茅台和腾讯
2. 计算相对买入价的盈亏比例
3. 输出持仓盈亏表格
4. 检测异动并提醒
